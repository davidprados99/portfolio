## sys.info [Description]

This repository contains the source code for my personal web portfolio. Built from scratch to deliver an immersive User Experience (UX), it emulates a legacy Command Line Interface (CLI) using monospace fonts, high-contrast color palettes (phosphor green on black), and terminal visual effects, without compromising modern accessibility and responsive design.

## Core Features

* **[ RETRO_INTERFACE ]**: Custom styling using Tailwind CSS v4 to simulate CRT monitors.
* **[ RESPONSIVE_DESIGN ]**: Seamless adaptation from mobile devices to ultrawide displays.
* **[ DYNAMIC_ROUTING ]**: Fast, reload-free navigation leveraging a Single Page Application (SPA) architecture.
* **[ SECURE_FORM ]**: Formspree API integration for direct email routing without exposing credentials.
* **[ AUTO_DEPLOY ]**: Continuous Integration/Continuous Deployment (CI/CD) via GitHub Actions to GitHub Pages.

## Local Installation

To boot this system in a local development environment, execute the following commands in your terminal:

```bash
# 1. Clone the repository
git clone [https://github.com/davidprados99/portfolio.git](https://github.com/davidprados99/portfolio.git)

# 2. Access the project directory
cd portfolio

# 3. Install dependencies
npm install

# 4. Boot the development server
npm run dev
